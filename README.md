Description
-----------

[Enunciate](http://enunciate.codehaus.org) modules.

Building
--------

### Requirements

* [Maven](http://maven.apache.org) 2+
* [Java](http://java.sun.com) 5+

Check-out and build:

    git clone git@github.com:jdillon/enunciate-modules.git
    cd enunciate-modules
    mvn install
