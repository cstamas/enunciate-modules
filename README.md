Description
-----------

[Enunciate](http://enunciate.codehaus.org) modules for [Apache Wink](http://incubator.apache.org/wink).

Building
--------

### Requirements

* [Maven](http://maven.apache.org) 2+
* [Java](http://java.sun.com) 5+

Check-out and build:

    git clone git@github.com:jdillon/enunciate-wink.git
    cd enunciate-wink
    mvn install
